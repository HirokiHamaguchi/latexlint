`.tex`ファイルの`^23`、`_23`、`^ab`、`_ab`などのケースを検出します。`{}`または空白を追加して上付き文字と下付き文字の範囲を明確にしてください。
このルールはデフォルトで無効です。

`\includegraphics{figure_23}`や`\url{http://example.com/abc_123}`のようなファイル名/URL/ラベルは無視されます。
