### LLLlGg

Detect `<<` and `>>` in `.tex` or `.md` files.
You should likely use `\ll` and `\gg` instead.

![rules/LLLlGg](rules/LLLlGg/LLLlGg.png)

We do not detect `<<` like this one.

```md
I like human $<<<$ cat $<<<<<<<$ dog.
```
