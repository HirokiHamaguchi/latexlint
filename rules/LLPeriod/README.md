Detect abbreviation periods in `.tex` files.
This rule checks `e.g.`, `i.e.`, `i.i.d.`, `w.r.t.`, and `w.l.o.g.` when followed by a space.
You should likely add a comma (e.g., `e.g.,`) or use `\ ` (e.g., `e.g.\ `) to avoid spacing issues.
