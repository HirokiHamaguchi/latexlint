`.tex`と`.md`ファイルの`\cap_`、`\cup_`、`\odot_`、`\oplus_`、`\otimes_`、`\sqcup_`、`uplus_`、`\vee_`、`\wedge_`を検出します。

代わりに`\bigcap`、`\bigcup`、`\bigodot`、`\bigoplus`、`\bigotimes`、`\bigsqcup`、`\biguplus`、`\bigvee`、`\bigwedge`を使用すべきです。
