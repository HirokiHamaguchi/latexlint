\section{LLSharp}

Some people mistake \verb|\sharp| to be a symbol for the number of elements in a set, but it is not.
It is a musical symbol.
The correct symbol for the number of elements in a set is \verb|\#|.

\begin{table}[H]
	\centering
	\begin{tabular}{lll}
		\verb|\#|     & $\#\{ x \mid x \geq 0 \}$                      & \tA{ok} \\
		\verb|\sharp| & \hspace{0.11em} $\sharp \{ x \mid x \geq 0 \}$ & \tD{ng} \\
	\end{tabular}
\end{table}

\begin{enumerate}[label=\textbullet]
	\item matching examples: $\sharp A$, $\sharp \{1,2,3\}$, $\sharp X_{0}$
	\item non-matching examples: ${\sharp}$, $f^\sharp X$, $\alpha_\sharp X$, $C\sharp$
\end{enumerate}

If you really want to use $\sharp$, you can disable this rule.
